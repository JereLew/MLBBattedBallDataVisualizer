# Batted Ball Data Visualizer for MLB

** Description
This application contextualizes & visualizes the data from `BattedBallData.xlsx`. 
It provides interactive scatter plots and summary statistics for various key columns.

** Installation
1. Ensure Python 3.x is installed.
2. Install/Use code editor: (ex: vscode)

** Steps to reproduce locally
1. Clone the Repository: Clone this repository to your local machine
2. Navigate to the the correct project directory: 'cd <repo>'
3. Develop virtual environment for dependencies: 'python -m venv venv'
  - MacOS/Linux: 'source venv/bin/activate'
  - Window: '.\venv\Scripts\activate'
4. Ensure all python packages are available: 'pip install -r requirements.txt'
5. Run app locally through terminal:
  - python webapp.py
  - Copy the url into browser: 'http://127.0.0.1:8050'
